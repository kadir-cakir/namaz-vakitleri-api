Namaz Vakitleri API

Diyanet İşleri Başkanlığı sitesindeki verileri kullanarak namaz vakitleri verileri sunar.
Ücretsiz sunucu kullanıldığından ilk açılış 10-15 saniye sürebilir. İlk açılıştan sonra hızlı çalışır.
Kaynak Kodlar (GitHub)

Kullanım

Ülkelerin listesi: /countries
Şehirlerin/Eyaletlerin listesi: /cities?country=2
İlçelerin/Bölgelerin listesi: /regions?country=2&city=503
Vakit verileri: /data?region=9541
